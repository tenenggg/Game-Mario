TUTORIAL UNTUK LAUNCH GAME NI
===============================

Game ni boleh dijalankan dalam 2 cara:
1. Run secara LOCAL di komputer sendiri
2. DEPLOY ke online supaya orang lain boleh main


════════════════════════════════════════════════════════════════
CARA 1: RUN GAME SECARA LOCAL (DI KOMPUTER SENDIRI)
════════════════════════════════════════════════════════════════

LANGKAH 1: Download & Install Node.js
--------------------------------------
1. Pergi ke website: https://nodejs.org/
2. Download versi LTS (Long Term Support) - disyorkan untuk pengguna biasa
3. Jalankan installer yang telah download
4. Klik "Next" sehingga selesai installation
5. Pastikan tick option "Add to PATH" semasa install
6. Restart komputer selepas install (optional tapi disyorkan)

LANGKAH 2: Download & Install Visual Studio Code (VS Code)
-----------------------------------------------------------
1. Pergi ke website: https://code.visualstudio.com/
2. Download VS Code untuk Windows
3. Jalankan installer
4. Klik "Next" sehingga selesai installation
5. Pastikan tick option:
   - "Add to PATH"
   - "Register Code as an editor for supported file types"
   - "Add 'Open with Code' action to Windows Explorer file context menu"
   - "Add 'Open with Code' action to Windows Explorer directory context menu"

LANGKAH 3: Buka Folder Game Dalam VS Code
------------------------------------------
1. Buka VS Code
2. Klik "File" > "Open Folder"
3. Cari dan pilih folder "game mario" (folder yang ada file-file game ni)
4. Klik "Select Folder"

LANGKAH 4: Buka Terminal Dalam VS Code
---------------------------------------
1. Dalam VS Code, klik menu "Terminal" > "New Terminal"
   ATAU tekan Ctrl + ` (backtick, sebelah kiri nombor 1)
2. Terminal akan muncul di bahagian bawah VS Code

LANGKAH 5: Install Dependencies (Package Yang Diperlukan)
----------------------------------------------------------
1. Dalam terminal yang dibuka tadi, taip command ni:
   
   npm install

2. Tekan Enter
3. Tunggu sehingga proses selesai (mungkin ambil masa 1-5 minit bergantung kelajuan internet)
4. Anda akan nampak banyak text muncul - itu normal
5. Bila dah selesai, anda akan nampak cursor berkedip semula

LANGKAH 6: Jalankan Game
-------------------------
1. Selepas npm install selesai, taip command ni dalam terminal:
   
   npm run dev

2. Tekan Enter
3. Anda akan nampak message macam ni:
   
   VITE v5.x.x  ready in xxx ms
   ➜  Local:   http://localhost:5173/
   ➜  Network: use --host to expose

4. Tekan Ctrl + Klik pada link "http://localhost:5173/"
   ATAU copy link tu dan paste dalam browser (Chrome, Firefox, Edge, etc.)

5. Game akan terbuka dalam browser!

LANGKAH 7: Stop Game
---------------------
1. Untuk stop game server, balik ke VS Code terminal
2. Tekan Ctrl + C
3. Kalau ada prompt "Terminate batch job (Y/N)?", taip Y dan Enter
4. Server akan stop dan game tak boleh access lagi

NOTA PENTING:
- Setiap kali nak main game, kena run "npm run dev" lagi
- Terminal kena dibiarkan running (jangan close) selagi nak main game
- Kalau ada error, pastikan semua langkah atas dibuat dengan betul
- Kalau masih error, cuba delete folder "node_modules" dan run "npm install" lagi


════════════════════════════════════════════════════════════════
CARA 2: DEPLOY GAME KE ONLINE (SUPAYA ORANG LAIN BOLEH MAIN)
════════════════════════════════════════════════════════════════

Untuk deploy game ke online, anda perlukan platform third-party.
Berikut adalah cara untuk deploy menggunakan platform popular (PERCUMA):


OPTION A: DEPLOY MENGGUNAKAN VERCEL (PALING SENANG & DISYORKAN)
----------------------------------------------------------------

LANGKAH 1: Create Account GitHub (Kalau Belum Ada)
1. Pergi ke https://github.com/
2. Klik "Sign Up"
3. Ikut langkah-langkah untuk create account
4. Verify email anda

LANGKAH 2: Upload Code Ke GitHub
1. Pergi ke https://github.com/
2. Login dengan account anda
3. Klik button "+" (atas kanan) > "New repository"
4. Bagi nama repository (contoh: mario-game)
5. Pilih "Public" (supaya free)
6. Klik "Create repository"
7. Ikut arahan yang diberi untuk upload code:
   - Buka terminal dalam folder game (dalam VS Code)
   - Taip command berikut satu-persatu:
   
   git init
   git add .
   git commit -m "first commit"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO-NAME.git
   git push -u origin main
   
   (Tukar USERNAME dengan username GitHub anda dan REPO-NAME dengan nama repo yang dibuat)

LANGKAH 3: Create Account Vercel
1. Pergi ke https://vercel.com/
2. Klik "Sign Up"
3. Pilih "Continue with GitHub"
4. Authorize Vercel untuk access GitHub account anda

LANGKAH 4: Deploy Game
1. Selepas login Vercel, klik "Add New..." > "Project"
2. Pilih repository game yang anda upload tadi (contoh: mario-game)
3. Klik "Import"
4. Vercel akan auto-detect yang ini adalah Vite project
5. Framework Preset: Vite (auto-selected)
6. Root Directory: ./ (default)
7. Build Command: npm run build (auto-filled)
8. Output Directory: dist (auto-filled)
9. Klik "Deploy"
10. Tunggu 1-3 minit untuk deployment selesai
11. Bila dah selesai, anda akan dapat URL (contoh: https://mario-game.vercel.app)
12. Copy URL tu dan share dengan orang lain untuk main game!

LANGKAH 5: Update Game (Kalau Ada Changes)
1. Buat perubahan dalam code
2. Dalam terminal, run:
   
   git add .
   git commit -m "update game"
   git push
   
3. Vercel akan auto-deploy changes anda dalam beberapa minit!


OPTION B: DEPLOY MENGGUNAKAN NETLIFY
-------------------------------------

LANGKAH 1: Create Account Netlify
1. Pergi ke https://www.netlify.com/
2. Klik "Sign Up"
3. Pilih "GitHub" untuk sign up
4. Authorize Netlify

LANGKAH 2: Deploy Via GitHub (Kena Setup GitHub Dulu - Rujuk Option A)
1. Klik "Add new site" > "Import an existing project"
2. Pilih "GitHub"
3. Pilih repository game anda
4. Build settings:
   - Build command: npm run build
   - Publish directory: dist
5. Klik "Deploy site"
6. Tunggu deployment selesai
7. Anda akan dapat URL untuk game (contoh: https://random-name.netlify.app)
8. Boleh customize URL dalam site settings


OPTION C: DEPLOY MENGGUNAKAN GITHUB PAGES (100% PERCUMA)
---------------------------------------------------------

LANGKAH 1: Setup GitHub Repository (Rujuk Option A untuk upload code)

LANGKAH 2: Install gh-pages Package
1. Dalam terminal VS Code, run:
   
   npm install --save-dev gh-pages

LANGKAH 3: Update package.json
1. Buka file package.json
2. Tambah line ni dalam section "scripts":
   
   "predeploy": "npm run build",
   "deploy": "gh-pages -d dist"

3. Tambah line ni atas sekali dalam package.json (after opening brace):
   
   "homepage": "https://USERNAME.github.io/REPO-NAME",
   
   (Tukar USERNAME dan REPO-NAME dengan yang betul)

LANGKAH 4: Update vite.config.js
1. Buka file vite.config.js
2. Tambah base URL:
   
   export default defineConfig({
     base: '/REPO-NAME/',  // Tambah line ni
     plugins: [react()],
   })
   
   (Tukar REPO-NAME dengan nama repository anda)

LANGKAH 5: Deploy
1. Dalam terminal, run:
   
   npm run deploy

2. Tunggu proses selesai
3. Game akan available di: https://USERNAME.github.io/REPO-NAME

LANGKAH 6: Enable GitHub Pages (First Time)
1. Pergi ke repository GitHub anda
2. Klik "Settings"
3. Klik "Pages" (sidebar kiri)
4. Source: Deploy from a branch
5. Branch: pilih "gh-pages" dan folder "/ (root)"
6. Klik "Save"
7. Tunggu beberapa minit, refresh page
8. Anda akan nampak link game anda!


════════════════════════════════════════════════════════════════
TROUBLESHOOTING (MASALAH BIASA & PENYELESAIAN)
════════════════════════════════════════════════════════════════

MASALAH 1: "npm: command not found" atau "'npm' is not recognized"
PENYELESAIAN:
- Node.js belum install atau belum register dalam PATH
- Install semula Node.js dan pastikan tick "Add to PATH"
- Restart komputer lepas install
- Cuba buka VS Code yang baru (close semua window dan buka balik)

MASALAH 2: "Error: EACCES" atau permission denied
PENYELESAIAN:
- Run VS Code sebagai Administrator
- Right click VS Code icon > "Run as administrator"

MASALAH 3: Port 5173 already in use
PENYELESAIAN:
- Ada instance lain game yang masih running
- Close semua terminal dan cuba lagi
- Atau specify port lain: npm run dev -- --port 3000

MASALAH 4: Page blank/putih lepas deploy
PENYELESAIAN:
- Check vite.config.js - pastikan base URL betul
- Check browser console (F12) untuk errors
- Pastikan build command dan output directory betul

MASALAH 5: Game berjalan local tapi tak jalan selepas deploy
PENYELESAIAN:
- Check file paths - kena gunakan relative paths
- Check console errors dalam browser (F12)
- Pastikan semua assets ada dalam folder public atau imported betul


════════════════════════════════════════════════════════════════
TIPS & BEST PRACTICES
════════════════════════════════════════════════════════════════

1. TESTING LOCAL DULU
   - Selalu test game local dulu sebelum deploy
   - Pastikan tiada errors dalam console

2. GUNAKAN GIT
   - Save progress regular dengan git commit
   - Senang nak rollback kalau ada masalah

3. ENVIRONMENT VARIABLES
   - Kalau ada API keys atau secrets, jangan hardcode
   - Guna environment variables (.env file)
   - Tambah .env dalam .gitignore

4. OPTIMIZE BEFORE DEPLOY
   - Compress images kalau besar sangat
   - Remove console.logs yang tak perlu
   - Test build local: npm run build && npm run preview

5. MONITOR DEPLOYMENT
   - Check deployment logs kalau ada error
   - Test deployed site thoroughly
   - Check pada different browsers dan devices
